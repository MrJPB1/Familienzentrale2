# Display Layout

## Fokus: Zentrale Familienanzeige

Die Familienzentrale ist für ein dauerhaft sichtbares Wanddisplay optimiert.

## Hauptlayout

```text
┌──────────────────────────────────────────────┐
│ Uhrzeit | Familie | Profile                │
├───────────────┬─────────────────────────────┤
│ Navigation    │ Wochenkalender             │
│               │                             │
│               │                             │
├───────────────┼─────────────────────────────┤
│ Heute         │ Aufgaben                   │
│ Essen         │ Einkauf                    │
│ Erinnerungen  │ Punkte                     │
└───────────────┴─────────────────────────────┘
```

## Designprinzipien

- große Touchflächen
- einfache Bedienung
- kinderfreundliche Icons
- klare Farben
- wenig Text
- schnelle Übersicht
- aus Distanz lesbar
- optimiert für 24/7 Nutzung

## Verhalten

### Standard

Immer Dashboard anzeigen.

### Nach Inaktivität

```text
60 Sekunden → zurück zum Dashboard
5 Minuten → Bilderrahmenmodus
```

### Bilderrahmenmodus

- Familienfotos
- motivierende Nachrichten
- ruhige Farben
- Uhrzeit optional
