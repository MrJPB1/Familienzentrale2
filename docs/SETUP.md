# Setup

## Voraussetzungen
- Node.js 20+
- npm
- Git

## Installation

```bash
cd backend
npm install
```

## Start

```bash
npm start
```

Danach im Browser öffnen:

```text
http://localhost:3000
```

## Entwicklungsmodus

```bash
npm run dev
```

## Windows IoT / Kiosk

Chrome im Kioskmodus:

```bash
chrome.exe --kiosk http://localhost:3000
```
