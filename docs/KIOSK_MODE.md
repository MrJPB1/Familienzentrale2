# Zentrales Display / Kioskmodus

Die Familienzentrale ist primär für ein dauerhaft aktives Wanddisplay optimiert.

## Zielgerät

- Windows 10 IoT
- Touchscreen
- Chrome im Vollbildmodus
- Dauerbetrieb

## Optimierungen

- große Touchflächen
- kinderfreundliche Bedienung
- große Schrift
- hoher Kontrast
- ruhiges Layout
- Familien-Dashboard als Startansicht
- automatische Rückkehr zur Hauptansicht
- Bilderrahmenmodus bei Inaktivität

## Start

### Server starten

```bash
start-server.bat
```

### Kiosk starten

```bash
start-kiosk.bat
```

## Chrome Einstellungen

Empfohlen:

- Vollbild
- keine Tabs
- kein Mauszeiger
- automatische Anmeldung
- Bildschirm niemals ausschalten

## Geplante Erweiterungen
